# Know-Thyself

A role-based academic growth and self-assessment platform built using Flask.

## Features
- Student self-assessment & reflection modules
- Teacher dashboards & analytics
- Growth Hub with qualitative insights
- Role-based access control
- Automated UI validation

## Tech Stack
- Flask (Python)
- MongoDB
- Tailwind CSS
- Selenium (for validation)

## Purpose
Built as an academic project for institutional innovation and award submission.
